### [GET] - Get user session data
```url copy
https://api.spalla.io/v1/analytics/sessions/:videoId/:uid
```

Return information about one user session, specified by `uid`. You should get `uid` by using the *player notifications feature.*

***Path variables:***
* `:uid` - <u>Required.</u> The unique identifier for the user you want details about.
* `:videoId` - The unique identifier for the video or live streaming you want details about.
