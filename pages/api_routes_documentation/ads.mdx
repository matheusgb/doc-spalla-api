the different API endpoints specifically designed for configuring and customizing VAST 
ad functionalities. Utilize these endpoints to enhance your application's ad serving 
capabilities and provide a seamless integration of VAST ads into your project. Enjoy the 
flexibility and control offered by these configuration endpoints to optimize your ad playback 
experience!

### [PATCH] - Update VAST settings
```url copy
https://api.spalla.io/v1/directories/:directory_id/settings
```

Modifies the VAST settings taking into account your url and a stipulated time.
<Callout type="info" emoji="ℹ️">
If a video already has more than 1 ad, and you want to delete one of those 2, use this endpoint sending only the link you want to keep.
</Callout>

***Path variables:***
* `:directory_id` - <u>Required.</u> The unique identifier for the directory you want change vast settings.

```json filename="SUCCESS - 200 OK"
{
    "ads": {
        "vast": [
            {
                "url": "https://example.com",
                "offset": "start"
            },
            {
                "url": "https://example.com",
                "offset": "00:50:00"
            },
            {
                "url": "https://example.com",
                "offset": "end"
            }
        ]
    }
}
``` 
---
### [DELETE] - Remove VAST settings
```url copy
https://api.spalla.io/v1/directories/:directory_id/settings
```

Excludes the VAST settings from the folder passed in the url.

***Path variables:***
* `:directory_id` - <u>Required.</u> The unique identifier for the directory you want change vast settings.

```json filename="BODY EXAMPLE"
{
    "ads": {
        "vast": null
    }
}
```
