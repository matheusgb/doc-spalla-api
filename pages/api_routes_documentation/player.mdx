### [GET] - Show player configuration
```url copy
https://api.spalla.io/v1/players/configuration/:videoId?uid=:uid
```

Returns the base assets to get Spalla Player up and running. You must use `assets.hls` in 
TheoPlayer TypedSource configuration.

***Parameters:***\
`:uid` - <u>Required.</u> Unique string or number used to identify the user downloading this file. 
Valid characters: `A-Za-z0-9._@-` Max length: `32` Min length: `1`.

***Path variables:***\
`:videoId` - The unique identifier for the video you want details about.
