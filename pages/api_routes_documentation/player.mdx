# Player endpoints
Here you will find the player endpoints. Below, you can explore the various API endpoints 
related to player functionalities. Use these endpoints to enhance your application's audio 
and video playback capabilities. Enjoy the seamless integration of player features into 
your project!

## Player configuration
import Requests from '../../components/Requests.tsx'

<Requests method="GET" url="v1/players/configuration/:videoId?uid=:uid" 
title="Show player configuration"/>

Returns the base assets to get Spalla Player up and running. You must use `assets.hls` in 
TheoPlayer TypedSource configuration.

***Parameters:***\
`:uid` - <u>Required.</u> Unique string or number used to identify the user downloading this file. 
Valid characters: `A-Za-z0-9._@-` Max length: `32` Min length: `1`.

***Path variables:***\
`:videoId` - The unique identifier for the video you want details about.
